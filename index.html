JsonJ

JsonJ is a framework for working with json in Java. It has a few nice and unique features that distinguishes it from other frameworks:
  • The core JsonJ classes for json arrays and json dictionaries extend the Java Collections framework.
  • Several methods are provided that make life easier when it comes to manipulating json objects. You can deepClone, deepEquals, extract and create using string labels, etc.
